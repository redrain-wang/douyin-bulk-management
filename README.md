# Douyin Console Bulk Management

## 功能简介
本扩展支持抖音后台视频列表的批量操作，提供"全选"和"批量删除"按钮，方便一键管理视频。

## 安装方法
1. 克隆或下载本仓库。
2. 打开 Chrome 浏览器，访问 `chrome://extensions`。
3. 开启右上角"开发者模式"。
4. 点击"加载已解压的扩展程序"，选择本项目中的 `chrome-extension` 文件夹。

## 使用方法
1. 进入抖音创作者服务平台（`https://creator.douyin.com/`）。
2. 进入视频管理页面。
3. 页面右上角会自动出现"全选"和"批量删除"按钮。
4. 点击"全选"可选中所有视频。
5. 点击"批量删除"可一键删除所有已选中的视频。

## 注意事项
- **删除操作不可撤销，请谨慎使用。**
- 删除请求会使用当前登录的抖音创作者账号。
- 首次使用前，请先在抖音创作者平台的控制台（F12）查看实际的DOM结构和API接口，可能需要根据实际情况调整代码。
- 由于抖音平台可能会更新页面结构和API，如果插件失效，需要更新相应的选择器和接口地址。

## 开发说明
本插件需要根据抖音创作者平台的实际情况进行调整：

### 需要确认的信息
1. **视频列表的DOM结构**：视频项的class名称（当前猜测：`.aweme-item`, `.video-card`等）
2. **视频ID的获取方式**：从哪里提取视频ID（data属性、URL、图片地址等）
3. **删除API接口**：实际的删除接口地址和请求参数格式

### 调试步骤
1. 打开抖音创作者平台视频管理页面
2. 按F12打开开发者工具
3. 在Elements标签查看视频列表的DOM结构
4. 在Network标签删除一个视频，观察请求的URL和参数
5. 根据实际情况修改 `content_unified.js` 中的相关代码
