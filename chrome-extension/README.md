# Douyin Bulk Management

抖音创作者平台批量管理工具，支持视频批量选择和删除操作。

## 功能特性

- ✅ 批量选择视频
- ✅ 批量删除视频
- ✅ 自动检测tab切换（已发布/审核中/未通过/私密）
- ✅ 按需加载视频列表，性能优化
- ✅ 支持滚动加载更多视频

## 使用方法

1. 安装插件后，访问 [抖音创作者平台](https://creator.douyin.com/)
2. 进入"内容管理"页面
3. 页面顶部会自动出现"全选"和"批量删除"按钮
4. 选择需要删除的视频，点击"批量删除"

## 安全说明

- 本插件仅在抖音创作者平台页面运行
- 不收集任何用户数据
- 所有操作都在本地浏览器中完成
- 删除操作需要用户确认
